# Fashion Price Monitor v2.2

Real links extraction + CLP conversion for Chilean fashion retailers.

## Features
✅ Real product links extracted from HTML
✅ Automatic price conversion to CLP (Chilean Pesos)
✅ Multiple fashion stores supported
✅ OWASP security compliant

## Supported Stores
- Nike
- Adidas
- H&M
- Puma
- Vans
- DC Shoes
- Bold
- Fashion Park

## API Endpoint
```
GET https://fashion-price-monitor.vercel.app/api/scraper?stores=nike
```

Response includes real links and CLP prices.

## Deployment
Automatically deployed on Vercel via GitHub sync.
